## Usage of the bot

### Commands

## Developers

If you want to run the bot yourself, [create a new Discord Bot](https://discordapp.com/developers/docs/intro#bots-and-apps) and copy the token into a file called `auth.json`. It should look like this:

```json
{
    "token": "[Your Token]"
}
```
